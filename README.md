# Proverbs31 Underwrite - Prototype

This repository contains the prototype for the Proverbs31 Underwrite platform, a real estate underwriting tool designed for efficiency and investor-ready screening.

## Features

*   **Dashboard:** Overview of active deals, LOIs sent, and key metrics.
*   **Deal Pipeline:** Visual representation of the underwriting process.
*   **My Deals:** List of all underwriting deals with filtering and sorting.
*   **New Deal Wizard:** Step-by-step process for uploading documents and starting the underwriting.
*   **Underwriting Workbench:** Interactive tool for quick inputs, financial calculations, and deal scoring.
*   **Settings:** Manage profile, company information, LOI templates, and integrations.

## Technologies Used

*   **HTML5/CSS3:** For structure and styling.
*   **Vanilla JavaScript:** For application logic and UI rendering.
*   **Chart.js:** For data visualization.
*   **XLSX:** For spreadsheet parsing.

## Getting Started

To view the prototype, simply open the `index.html` file in any modern web browser.
